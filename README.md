# Projet Le Blog de Batman

### Cloner le projet

```
git clone https://github.com/vincent110421/leblogdebatman.git

### Déplacer le terminal dans le dossier cloné
```
cd leblogdebatman
```

### Installer les vendors (pour recréer le dossier vendor)
```
composer install
```

### Lancer le serveur
```
symfony serve
```
